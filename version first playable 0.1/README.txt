Español.

Requisitos:
*Windows 10
*Linux. (algunas versiones)
*Python 38 o superior
*Modulo Pygame 2.0 o superior
*Modulo Pillow 8.0 o superior
*Modulo Screeninfo

Para empezar a jugarlo abre la carpeta llamada "version first playable 0.1" y dentro de esta abre el archivo "main.py".

Controles:

*Flecha Derecha --> Mover hacia la derecha
*Flecha Izquierda --> Mover hacia la izquierda
*Flecha Arriba --> Saltar
*n --> Aumentar la velocidad.*NOTA: esta tecla es solo de prueba y no estara en el juego final, a veces hay que pulsarla repetidas veces para que se pueda activar y asi mismo para desactivarla.

English.

Requirements:
* Windows 10
* Linux. (some versions)
* Python 38 or higher
* Pygame 2.0 module or higher
* Pillow Module 8.0 or higher
* Screeninfo module

To start playing it, open the folder called "version first playable 0.1" and inside this open the file "main.py".

Controls:

* Right Arrow -> Move Right
* Left Arrow -> Move Left
* Up Arrow -> Jump
* n -> Increase speed * NOTE: this key is only for testing and will not be in the final game, sometimes you have to press it repeatedly so that it can be activated and also to deactivate it.
